No Phish

Chrome Extension

No Phish is a Google Chrome extension which scans e-mails for suspicious indicators to warn about possible phishers. Suspicious e-mail addresses can be flagged via the extension, which creates a growing, increasingly accurate account of threatening phishers. Challenging backend development for Ellehacks 2020 with a team of 4.
